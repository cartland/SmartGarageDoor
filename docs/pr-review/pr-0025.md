# PR #25: ci: Run release AAB build on PRs

**Goal:** Remove push-only condition from `build_release` job so it also runs on PRs. Catches release-breaking changes before merge. Previously only `test` and `build_debug` ran on PRs — a PR could pass CI but break the release build.
