# PR #284: refactor: Replace SnoozeRepository StateFlow with observeSnoozeState() Flow

**Goal:** Last ADR-013 violation. `SnoozeRepository.snoozeState: StateFlow<SnoozeState>` → `fun observeSnoozeState(): Flow<SnoozeState>`. ViewModel converts to StateFlow via `stateIn(Eagerly, SnoozeState.Loading)`. After this + #282 + #283, zero StateFlow below ViewModel layer.

**Notes:** Later reverted by ADR-022 (PR #360) which moved StateFlow ownership back to the repository — the rule here was wrong in its direction.
