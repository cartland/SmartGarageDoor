# PR #206: fix: Block git push to branches with auto-merge enabled

**Goal:** Add PreToolUse hook check: before `git push`, query if target branch's PR has auto-merge enabled. If active, push blocked with instructions to disable auto-merge first. Prevents race condition where squash merge executes before new commits arrive, silently losing them.
