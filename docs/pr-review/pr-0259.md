# PR #259: feat: SquareButtonWithProgress custom Layout

**Goal:** Replace nested Column + `widthIn` + `aspectRatio` with custom `Layout` that guarantees square button, progress bar matches button width exactly, single measurement pass (no view hopping on first frame), adapts to bounded/unbounded constraints. Evaluated 4 alternatives (IntrinsicSize.Min, BoxWithConstraints, onSizeChanged+state, custom Layout) — only custom Layout actually solves the problem.
