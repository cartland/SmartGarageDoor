# PR #34: ci: Add test coverage enforcement Gradle task

**Goal:** Custom `buildSrc` Gradle task (`checkTestCoverage`) that fails build if ViewModel or Repository classes lack test files. Scans `*ViewModel` and `*Repository` classes. Matches `*Impl` to base-name tests. Skips interfaces, Hilt modules, enums, abstract classes. Central exemptions in `test-coverage-exemptions.txt`. Inline exemptions via `// @NoTestRequired: reason`. 6 classes exempt (Firebase wrapper).

**Files:** `AndroidGarage/buildSrc/src/main/kotlin/testcoverage/TestCoverageCheckTask.kt` (new, 133 lines), `buildSrc/build.gradle.kts`, `build.gradle.kts`, `test-coverage-exemptions.txt` (new).
