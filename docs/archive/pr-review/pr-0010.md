# PR #10: refactor: Add DispatcherProvider interface for injectable coroutine dispatchers

**Goal:** Add `DispatcherProvider` interface with `main`, `io`, `default` dispatchers. `DefaultDispatcherProvider` (production) bound via Hilt. `TestDispatcherProvider` for tests. Update all ViewModels (DoorViewModelImpl, RemoteButtonViewModelImpl, AuthViewModelImpl, AppLoggerViewModelImpl) to inject `DispatcherProvider` instead of using `Dispatchers.IO` directly.
